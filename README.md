# avito-shop

Сервис для внутреннего магазина мерча, где сотрудники могут приобретать товары за монеты. Кроме этого монеты можно передавать другим сотрудникам.

## Функциональность

1. JWT-аутентификация
2. Возможность купить мерч
3. Возможность перевести монеты другим сотрудникам в виде благодарности или подарка
4. Сохранение всех покупок и переводов монет (транзакций)

## Запуск
Для запуска используйте Docker.

Запустить сервис

```bash
docker build .
docker compose --profile run up -d
```

После запуска сервис доступен по адресу: http://localhost:8080

Запустить тесты

```bash
docker build .
docker compose --profile test up --build --abort-on-container-exit -d
```
